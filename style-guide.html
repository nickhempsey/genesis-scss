IngredientSAFE will be using Bootstrap v4.5 with augmentations to color scheme, fonts, and several other pieces to make it come into line with the brand.

For more information on Bootstrap v4.5

In addition, we will be using:

  • FontAwesome for icons
  • Swiper for carousels
  • Photoswipe for image gallery overlays
  • GravityForms for Forms
